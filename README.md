# 🚪 IoT MyDoor - Điều khiển cửa cuốn qua ESP32 & MQTT

Hệ thống điều khiển cửa cuốn từ xa qua điện thoại, sử dụng giao thức MQTT để đảm bảo tốc độ phản hồi nhanh và tính ổn định cao.

## 🛠 Linh kiện sử dụng
- **Vi điều khiển:** ESP32 (WROOM-32).
- **Module điều khiển:** Relay 2 kênh (Điều khiển nút Lên/Xuống).
- **Giao thức:** MQTT (Broker free).
- **Ứng dụng:** MyMQTT trên Smartphone.

## 🚀 Tính năng chính
- Điều khiển cửa Đóng/Mở từ bất kỳ đâu có Internet.
- Phản hồi trạng thái kết nối của thiết bị (Status LED).
